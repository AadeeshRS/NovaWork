## ğŸ“š Quick Definitions

- **Class**: Blueprint for creating objects
- **Object**: Instance of a class with actual data
- **Constructor**: Special method that runs when creating object
- **Static**: Property/method shared by all class instances
- **this**: Reference to current object instance
- **Method**: Function inside a class
- **JSON**: JavaScript Object Notation, data format
- **Async**: Function that runs asynchronously (doesn't block)
- **Await**: Pauses execution until promise resolves
- **Fetch**: Gets data from a file/URL
- **Parse**: Convert text to JavaScript object
- **DOM**: Document Object Model, webpage structure
- **createElement**: Create new HTML element
- **appendChild**: Add element as child of another
